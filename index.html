欢迎来到Ionic中文站

Ionic是当代最好的Hybrid App开发框架,一个用HTML, CSS 跟JS 开发的一个用于移动设备的web app 开发框架，采用 Sass与AngularJS 开发。
Ionic提供了精美的类原生应用的用户体验，像下拉刷新，无限加载等，使我们可以专注在业务方面。 Ionic cli命令行工具集成了cordova,crosswalk等高级工具，可以让你轻松建立Hybrid App。

开启你的Ionic之旅

$ npm install -g cordova ionic
$ ionic start myApp tabs
$ cd myApp
$ ionic platform add ios
$ ionic build ios
$ ionic emulate ios

Ionic中文站不仅提供Ionic的最新动态，更会提供大量的实例让你参考，让你更快，更好的学习Ionic，将Ionic应用到你的业务中。
同时，希望你学有所成之时，贡献你的作品，让更多的人学习。
